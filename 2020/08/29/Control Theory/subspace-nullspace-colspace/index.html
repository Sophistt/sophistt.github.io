Subspace, Null Space and Col Space in Linear Algebra

Created2020-08-29|Updated2023-02-19|StudyMath
|Word count:791|Reading time:4min|Post View:

subspace

A subspace of a vector space VVV is a subset HHH of VVV that has two properties.

  1. HHH is closed under vector addition. That is, for each uuu and vvv in HHH, the sum u+vu+vu+v is in HHH.
  2. HHH is closed under multiplication by scalars. That is, for each uuu in HHH and each scalar ccc, the vector cucucu is in HHH.

Note

In some materials, HHH has the third property that The zero vector of VVV is in HHH. This property could be inclued in Property 2 while c=0c=0c=0.

For example,

  • A line through the origin in R2R^2R2 is a subspace.

  • The xyxyxy-plane in R3R^3R3 is a subspace.

  • The set of all polynomials PPP is a subspace of C[0,1]C[0,1]C[0,1].

Proof

如何证明一个空间是subspace？

只需要根据定义，将该空间的向量 x,yx, yx,y 代入两个条件验证即可。

Span（线性生成空间）is the set of all linear combination of specialized vectors.

For example,

Null space

Given a matrix A∈ℜm×nA \in \Re^{m \times n}A∈ℜm×n. The null space or kernel of AAA is defined as

NulA=KerA={x∈ℜn∣Ax=0}Nul A = Ker A = \{ x \in \Re^n | Ax = 0\} NulA=KerA={x∈ℜn∣Ax=0}

Column space

Given a matrix A∈ℜm×nA \in \Re^{m \times n}A∈ℜm×n. The column space or image of AAA is defined as the set of all linear combination of the colums of AAA.

colA=ImA={b∈ℜm∣b=Axforsomex∈ℜn}col A = ImA=\{b \in \Re^m | b = Ax \quad \mathrm{for \quad some} \quad x \in \Re^n\} colA=ImA={b∈ℜm∣b=Axforsomex∈ℜn}
